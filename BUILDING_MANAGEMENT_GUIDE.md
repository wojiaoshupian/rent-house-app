# 🏢 楼宇管理入口使用指南

## 🎯 功能概述

楼宇列表入口已成功集成到楼宇管理系统中，提供便捷的导航路径。

## 🚀 导航路径

### 主要入口：楼宇管理
**路径**: 首页 → 楼宇管理 → 楼宇列表

**操作步骤**:
1. 打开应用，进入首页（MainHome）
2. 在"快捷操作"区域找到"楼宇管理"卡片
3. 点击"楼宇管理"卡片
4. 直接进入楼宇列表页面

### 辅助入口：Home页面
**路径**: Home页面 → 查看全部 → 楼宇列表

**操作步骤**:
1. 从首页导航到Home页面
2. 滚动到"楼宇列表"区域
3. 点击"查看全部"按钮
4. 进入楼宇列表页面
